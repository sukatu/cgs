# Images Directory

## Folder Structure

- `gallery/event-1/` - Images for CGS Accra 2025: Inaugural Series
- `gallery/event-2/` - Images for Webinar Series: Directorship in the Digital Age
- `gallery/event-3/` - Images for Stakeholder Networking Sessions

## How to Add Images

1. Download images from the Dropbox folder
2. Place them in the appropriate event folder
3. Name them sequentially (e.g., `event1-01.jpg`, `event1-02.jpg`, etc.)
4. Recommended formats: JPG, PNG
5. Recommended size: 1200x800px or larger for best quality

## Image Naming Convention

- Event 1: `event1-01.jpg`, `event1-02.jpg`, etc.
- Event 2: `event2-01.jpg`, `event2-02.jpg`, etc.
- Event 3: `event3-01.jpg`, `event3-02.jpg`, etc.

The pictures page will automatically display images once they are placed in these folders.
